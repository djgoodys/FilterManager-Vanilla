Adding new units

After logging in from the top of the page the navigation bar will appear. Click on the "Add new unit" button.

The add new unit page appears as seen below.

Fill in the unit specifications such as name, area served, location etc...

  • Unit Name
    • The name given to the unit. i.e.(AH-21C or Ware House A/C).
  • Location
    • This is the physical location of the unit.
  • Area Served
    • This is the area that the Unit serves. i.e.(buffet dining rm or security office).
  • Filter Size
    • There can be up to 2 different air filter sizes inputed for each unit.
  • Filter Type
    • Simply select the type of air filter from the drop down list. i.e. (Paper, 12 month extended, etc...)
  • Filter due date
    • The date at which the next air filter replacement is to be performed.
  • Filter Rotation
    • Input the number of months between air filter replacments. i.e. (The number 1 would be a filter replacement every month, 3 would be every 3 months or 12 would be once a year, 24 would be every 2 years.)
  • Belts
    • This is the size of the indoor blower belt if the unit uses belts and is not a direct drive.
  • Notes
    • If you wish to add any information about unit. i.e.(Coil last cleanded on 12/04/2022 or This unit requires a 14 ft. step latter to access filters).
Now click on the Add new unit button to save unit to data base.


Using Tasks


One purpose of using the "tasks" option is grouping units in your name so that others in your team can see that those units are asigned to you. Tasks can be used for 1 or more units at a time. If you are changing filters on many units and wish to group them they will show up on your tasks page. After changing the filters on a unit in your tasks you complete each task.

Adding a task:


To add a unit to your tasks list check the box next to the unit name in the Unit List
      BACK TO TOP


Viewing your task list:


Click on the Tasks button in the navigation bar.

Tasks list will appear on page like below.



Completing or Canceling a task:


Cancel a task by clicking on the cancel task icon.
Complete a task by clicking on the complete task icon.
Completing a task will also update your filter inventory automatically.

Viewing your task history.


You can view your completed task history by clicking on the Task History button.
